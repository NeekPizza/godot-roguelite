# Asset Credits & Licenses

Every asset committed to this repo MUST have an entry here recording source and
license. We ship commercially, so only CC0 / public-domain / self-generated
assets are permitted. No exceptions.

## Art

**None, by design.** Every visual — player, all five enemy types, projectiles,
XP gems, particles, world grid and walls — is drawn procedurally in-engine with
`_draw()` calls. There is no image file in this repository and therefore no art
licence to honour.

## Music

_None yet._

## SFX

All seven sounds are **generated by `tools/gen_sfx.py`** — original work created
for this project. No third-party asset, no attribution requirement, no licence
to verify. The generator is fixed-seeded, so re-running it reproduces the same
files byte for byte.

- **Files:** `assets/sfx/{shoot,hit,enemy_death,xp_pickup,level_up,player_hurt,run_over}.wav`
  **Source:** generated by `tools/gen_sfx.py` in this repo
  **Author:** this project (original work)
  **License:** owned outright — safe for commercial release
  **Verified:** 2026-07-18 — synthesised from scratch with the Python standard
  library; no sample material of any kind was used as input.

# Release checklist

Work top to bottom. Anything that fails stops the release — none of these are
advisory.

---

## 1. Verify the build

```bash
godot --headless tests/balance_test.tscn                              # data integrity
godot --headless tests/steering_test.tscn                             # input resolution
godot --headless tests/input_test.tscn                                # controller bindings
godot --headless tests/save_test.tscn -- --save-file=test_save.json   # ranked ledger + table
./tools/determinism_check.sh                                          # daily-seed determinism
```

All five exit non-zero on failure. The determinism check is the one that matters
most: **if it fails, the daily leaderboard is meaningless** and the game's whole
premise is broken. Do not ship past a red determinism run.

Then play a real run and confirm: menu → ranked confirmation → run → death →
score table → back to menu, plus pause and settings mid-run.

## 2. Confirm no console errors

```bash
godot --headless -- --date=2026-01-01 --max-seconds=120 --auto-pick --quit-on-end
```

Expect zero `ERROR` and zero `WARNING` lines.

Known non-issue: `--quit-after` (as opposed to `--quit-on-end`) reports
"resources still in use at exit". That is a forced-frame-kill artifact, not a
leak — see `docs/TESTING.md`.

## 3. Integrity checks specific to a leaderboard game

- [ ] **Test hooks cannot produce a ranked score.** `--godmode`, `--auto-pick`,
      `--time-scale`, `--max-seconds` and `--scripted-input` ship in the release
      binary (Godot passes user args to any build), so `RunConfig.begin_run()`
      refuses to run ranked when any is present. Verify:
      ```bash
      godot -- --ranked --godmode --max-seconds=5 --auto-pick --quit-on-end
      ```
      must print "test hooks active -> forced to practice" and leave the ranked
      ledger empty.
- [ ] **The ranked attempt is charged at run start**, so quitting mid-run grants
      no retry. Start a ranked run, kill the process, relaunch: the menu must
      show the attempt as used.
- [ ] **Scores are bounded** before storage and submission (`Score.bounded`).
- [ ] Anti-cheat beyond this is deliberately out of scope (GDD section 10).
      Local scores are editable and Steam submissions are client-trusted. Do not
      spend release time here.

## 4. Content and licensing

- [ ] Every asset in `assets/` has an entry in `assets/CREDITS.md` naming its
      source and licence. **No exceptions** — we ship commercially.
- [ ] All current assets are original work generated by `tools/gen_sfx.py` and
      `tools/gen_music.py`; there is no third-party art or audio, and no image
      file in the repository at all.
- [ ] If GodotSteam has been added (Phase 4): record that **GodotSteam is MIT
      but the bundled `steam_api` binaries are Valve's**, redistributable only
      under the Steamworks SDK Access Agreement.

## 5. Export

Export templates must match the engine version exactly:

```bash
godot --version                       # must equal the installed template version
```

Install templates via the editor (Editor → Manage Export Templates) or drop the
`.tpz` in `~/Library/Application Support/Godot/export_templates/<version>/`.
They are roughly 1 GB and are NOT in this repository.

```bash
godot --headless --export-release "Windows Desktop" build/windows/DailySeed.exe
```

- [ ] Export completes with no errors.
- [ ] **Run the exported build on actual Windows.** Exporting successfully is
      not evidence that it runs; a macOS export test proves nothing about the
      Windows binary.
- [ ] Confirm `tests/`, `tools/`, `docs/` and `build/` are excluded from the
      shipped package (see `exclude_filter` in `export_presets.cfg`).
- [ ] Confirm the game runs on a machine **without** Godot installed.

## 6. Steam (Phase 4 — not yet built)

Deferred deliberately until there is a real App ID: the leaderboard is the core
hook and writing it blind against Spacewar (480) would mean rebuilding it later.

- [ ] Steam Direct fee paid, app created, App ID issued.
- [ ] GodotSteam **v4.20.1-gde** prebuilt GDExtension from
      `codeberg.org/godotsteam/godotsteam` — not the module build, not a source
      build, and no Godot downgrade.
- [ ] Commit only the `osx` and `win64` binaries (~21 MB of the 97 MB archive).
- [ ] **Every Steam call guarded**: the game must run normally with Steam
      absent, closed, or offline. Test with the Steam client not running.
- [ ] Leaderboard submit and fetch verified against the real app.
- [ ] Achievements verified to unlock.
- [ ] App ID lives in config, never hardcoded at a call site.

## 7. Store page

- [ ] The daily seed rolls over at **00:00 UTC**, which is mid-evening in the
      Americas. Say so on the store page — players will otherwise think the
      daily changed at the wrong time.
- [ ] Capsule art, trailer/GIF of a late-game run, and a short description
      leading with the daily-seed hook.
- [ ] Price set ($1–5 band).

## 8. Tag the release

```bash
git tag -a v1.0.0 -m "1.0.0"
git push origin v1.0.0
```
